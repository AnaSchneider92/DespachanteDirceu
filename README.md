🚗 Despachante Dirceu – Site Institucional

Site institucional desenvolvido para apresentar os serviços do Despachante Dirceu e facilitar o contato com clientes de forma rápida, simples e eficiente.

🔗 Acesse o repositório:
https://github.com/AnaSchneider92/DespachanteDirceu

📌 Sobre o projeto

Este projeto consiste em uma aplicação web estática com navegação por abas, desenvolvida com foco na organização das informações e na experiência do usuário.

O site permite que clientes visualizem os serviços oferecidos e entrem em contato diretamente via WhatsApp.

✨ Funcionalidades
📄 Página inicial com apresentação do negócio
📂 Navegação por abas (Início, Serviços, Contato, etc.)
📱 Integração com WhatsApp para atendimento rápido
🎯 Interface simples e intuitiva
💻 Layout responsivo
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript
🚀 Como executar o projeto
Clone o repositório:
git clone https://github.com/AnaSchneider92/DespachanteDirceu.git
Acesse a pasta do projeto:
cd DespachanteDirceu
Abra o arquivo index.html no navegador.
